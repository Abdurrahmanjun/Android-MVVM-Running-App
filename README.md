# 🏃‍♂️ RunningApp – Open Source Run Tracking App

A modern, lightweight, and open-source run tracking application built with Android's latest technologies and best practices.

---

## 🔧 Tech Stack & Architecture (Work in Progress)

- **Min SDK:** 24  
- **Language:** 100% Kotlin  
- **Async:** Kotlin Coroutines & Flow  
- **Architecture:** MVVM with clean separation of concerns  
- **UI Toolkit:** Jetpack Compose  
- **Design:** Material Design 3 + custom animations  

---

## 🧩 Core Components & Libraries

### 📱 UI & Navigation
- **Jetpack Compose** – Declarative UI framework  
- **Navigation Compose** – For seamless screen navigation  
- **Hilt Navigation Compose** – Dependency injection across navigation graphs  
- **Material Design** – Google’s modern UI/UX principles  

### ⚙️ Architecture & DI
- **ViewModel** – Lifecycle-aware business logic container  
- **Hilt** – Dependency Injection by [Dagger](https://dagger.dev/hilt/)  

### 🗄️ Data Layer
- **Room** – Local database with SQLite backend  
- **Retrofit2** – Networking and REST API integration  
- **OkHttp3** – HTTP client with logging, interceptors, and mock server support  
- **Moshi** – Modern JSON serialization/deserialization for Kotlin  

### 📸 Media & Logs
- **Glide** – Efficient image loading and caching  
- **Timber** – Lightweight and extensible logging utility
